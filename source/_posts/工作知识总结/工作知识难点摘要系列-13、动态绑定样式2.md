# 一、动态绑定样式(补充)

## 1、简单介绍
<p>&emsp;&emsp;上次写了个动态样式绑定写的太简单了，后来发现还有一些比较有意思的场景没有考虑到，现在给他补充一下的。好的，那么我们接下来开始吧！！！！！！</p>

## 2、多种方式
### 2.1、普通对象动态添加（比较常见）

```javascript
<template>
	<view>
    	/* 动态添加颜色 */
		<view :style="{color:fontColor}"> </view>
		/* 2.动态添加边距 */
		<view :style="{ paddingTop: num + 'px' }"></view>
		/* 3.动态添加背景图片 */
		<view :style="{backgroundImage: 'url(' + imageURL + ')','background-repeat':'no-repeat', 'backgroundSize':'100% 100%'}"></view>
	</view>
</template>
 
<script>
	export default {
		data() {
			return {
				imageURL: 'https://app-file.beitaichufang.com/img/H5/web/banner/banner23.jpg', //图片
				num:20, //字体大小
				fontColor:'red' //字体颜色
			}
		}
	}
</script>
```

### 2.2、数组对象动态添加（加多个样式，有点意思）
```javascript
<template>
	<view>
    	/* 1.动态添加颜色，动态添加边距 */
		<view :style="[{paddingTop: num + 'px'},{color:fontColor}]"></view>
    	/* 2.动态添加背景图片  */
		<view :style="[{'background-image':`url(${imageURL})`},{'background-repeat':'no-repeat'},{'background-size':'100% 100%'}]"></view>
	</view>
</template>
 
<script>
	export default {
		data() {
			return {
				imageURL: 'https://app-file.beitaichufang.com/img/H5/web/banner/banner23.jpg', //图片
				num:20, //字体大小
				fontColor:'red' //字体颜色
			}
		}
	}
</script>
```

### 2.3、三目运算动态添加
```javascript
<template>
	<view>
    	/* 如果flag为true   颜色就为red色 */
		<view :style="[{color:(flag?fontColor:'green')},{fontSize:'20px'}]">green</view>
		<view :style="[{color:(!flag?fontColor:'green')}]">red</view>
		/* 如果flag为false  颜色就为green色 */
    	<view :style="[flag?{top:searchTop + 'px',width:searchWidth + 'px'}:{top:'100px',width:'100%'}]"></view>
	</view>
</template>
 
<script>
   export default {
		data() {
			return {
				fontColor:'red',
				flag:false,
        		searchTop:20,
        		searchWidth:100
		    }
	    }
    }
</script>
```

### 2.4、动态设置class
```javascript
<template>
     <view :class="flag?'flaGreen':'flagRed'">I's Xq</view>
</template>

<script>
   export default {
		data() {
			return {
				flag:false
			}
	    }
    }
</script>
<style>
	.flaGreen{
		color: green
	}
	.flagRed{
		color: red
	}
</style>
```
